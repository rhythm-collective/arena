# Arena

Connected arena.

## Setup

1. Run `pip install -e` from root.
2. Run `arena` to start arena. *(This currently does nothing.)*
3. Run `pytest` to execute full test suite.

## Mongodb Samples

- Note that MongoDB sample will fail unless you have set up an atlas instance.
